# Сайт-визитка репетитора по математике

Современный веб-сайт для репетитора по математике с системой личных кабинетов для учителя и учеников.

## Технологии

- **Next.js 15+** - React фреймворк с App Router
- **TypeScript** - Типизация
- **Prisma** - ORM для работы с PostgreSQL
- **NextAuth.js** - Аутентификация
- **shadcn/ui** - UI компоненты на основе Tailwind CSS
- **PostgreSQL** - База данных
- **Zod** - Валидация данных

## Функционал

### Публичная часть
- Лендинг с информацией о репетиторе
- Секции: Обо мне, Тарифы, Как проходят занятия, Контакты
- Форма обратной связи

### Личный кабинет учителя
- Расписание (календарный вид)
- Управление учениками
- Успеваемость с графиками
- Домашние задания
- Материалы для обучения
- ОГЭ: материалы и статистика пробников
- Управление оплатами
- Общая статистика

### Личный кабинет ученика
- Расписание уроков
- Домашние задания (просмотр и загрузка решений)
- Материалы для обучения
- ОГЭ: материалы и личная статистика

## Установка и запуск

### Требования
- Node.js 20+
- PostgreSQL 16+
- npm или yarn

### Локальная разработка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd nastya-site
```

2. Установите зависимости:
```bash
npm install
```

3. Настройте переменные окружения:
```bash
cp .env.example .env
```

Отредактируйте `.env` и укажите:
- `DATABASE_URL` - строка подключения к PostgreSQL
- `NEXTAUTH_SECRET` - секретный ключ (сгенерируйте через `openssl rand -base64 32`)
- `NEXTAUTH_URL` - URL приложения (для локальной разработки: `http://localhost:3000`)

4. Инициализируйте базу данных:
```bash
npx prisma generate
npx prisma db push
```

5. Запустите сервер разработки:
```bash
npm run dev
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)

## Скрипты

- `npm run dev` - Запуск сервера разработки
- `npm run build` - Сборка для продакшена
- `npm run start` - Запуск продакшен сервера
- `npm run lint` - Проверка кода линтером
- `npm run db:generate` - Генерация Prisma Client
- `npm run db:push` - Применение изменений схемы к БД
- `npm run db:migrate` - Создание миграции
- `npm run db:studio` - Открытие Prisma Studio
