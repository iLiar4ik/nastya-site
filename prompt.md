# Промпт для разработки сайта учителя математики Настя

## Общее описание проекта

Создать современный, профессиональный и привлекательный сайт для учителя математики Настя. Сайт должен представлять услуги репетиторства по математике, вызывать доверие у потенциальных клиентов и мотивировать их к записи на занятия.

## Целевая аудитория

- Ученики 5-11 классов, нуждающиеся в помощи по математике
- Родители, ищущие квалифицированного репетитора для своих детей
- Выпускники, готовящиеся к ОГЭ и ЕГЭ по математике
- Студенты первых курсов, нуждающиеся в закреплении школьной программы

## Основные блоки сайта

### 1. Главный блок (Hero Section)

**Содержание:**
- Фотография учителя (файл: /image/IMG_0967.JPG) - качественное профессиональное фото
- Заголовок: "Настя - профессиональный репетитор по математике"
- Подзаголовок: "Помогу вашему ребенку полюбить математику и сдать экзамены на высокие баллы"
- Ключевые преимущества (3-4 пункта):
  * Индивидуальный подход к каждому ученику
  * Подготовка к ОГЭ и ЕГЭ с высоким процентом поступления
  * Современные методики преподавания
  * Гибкий график занятий (онлайн и оффлайн)
- Призыв к действию: кнопка "Записаться на бесплатное пробное занятие"

**Дизайн:**
- Фон: светлый, минималистичный, возможно с ненавязчивыми математическими элементами
- Фотография слева или по центру, текст справа или под фото
- Акцентный цвет: синий или бирюзовый (ассоциируется с логикой, спокойствием)
- Шрифты: современный, легко читаемый заголовок и основной текст

### 2. Блок тарифных планов

**Содержание:**
Три тарифных плана в виде карточек:

**Начальный (для 5-7 классов):**
- Название: "Начальный курс"
- Цена: "1500 руб/60 мин"
- Описание: "Основы математики для младших классов"
- Преимущества:
  * Закрепление школьной программы
  * Развитие логического мышления
  * Помощь с домашними заданиями
- Кнопка: "Выбрать тариф"

**Стандартный (для 8-10 классов):**
- Название: "Стандартный курс" 
- Цена: "1800 руб/60 мин"
- Описание: "Углубленное изучение математики для средних классов"
- Преимущества:
  * Подготовка к ВПР и контрольным
  * Устранение пробелов в знаниях
  * Развитие навыков решения задач
- Кнопка: "Выбрать тариф"

**Продвинутый (для 11 классов и выпускников):**
- Название: "Подготовка к экзаменам"
- Цена: "2200 руб/60 мин"
- Описание: "Интенсивная подготовка к ОГЭ и ЕГЭ"
- Преимущества:
  * Разбор типовых экзаменационных заданий
  * Стратегия сдачи экзамена
  * Пробные тесты с разбором ошибок
- Кнопка: "Выбрать тариф"

**Дизайн:**
- Карточки в один ряд на десктопе, в столбик на мобильных устройствах
- Активный тариф может быть подсвечен акцентным цветом
- Четкое визуальное разделение тарифов
- Цена выделена крупным шрифтом

### 3. Блок информации об учителе

**Содержание:**
- Заголовок: "Обо мне"
- Фотография учителя (можно использовать то же фото или другое)
- Образование:
  * Высшее математическое образование (указать конкретный вуз)
  * Специализация по методике преподавания математики
  * Повышение квалификации (курсы, сертификаты)
- Методика преподавания:
  * Индивидуальный подход к каждому ученику
  * Использование современных образовательных технологий
  * Практико-ориентированное обучение
  * Регулярная обратная связь и прогресс-трекинг
- Достижения:
  * Процент учеников, сдавших ОГЭ/ЕГЭ на высокие баллы
  * Годы преподавательского опыта
  * Благодарные отзывы учеников и родителей
  * Участие в профессиональных конференциях

**Дизайн:**
- Двухколоночная структура: фото слева, текст справа (на мобильных устройствах фото сверху, текст снизу)
- Акцент на ключевых достижениях (возможно использование иконок или цифр)
- Текст разбит на понятные блоки с подзаголовками

## Технические требования

**Технологии:**
- HTML5 семантическая разметка
- CSS3 с использованием Flexbox/Grid для адаптивности
- JavaScript для интерактивных элементов (форма обратной связи, калькулятор стоимости)
- Оптимизация для мобильных устройств (Mobile First подход)

**Производительность:**
- Быстрая загрузка сайта (оптимизация изображений, минификация CSS/JS)
- SEO-оптимизация: семантическая разметка, мета-теги, alt-атрибуты для изображений
- Кроссбраузерная совместимость (Chrome, Firefox, Safari, Edge)
- Доступность (WCAG 2.1 AA уровень)

**Функциональность:**
- Форма обратной связи для записи на пробное занятие
- Интерактивный калькулятор стоимости занятий
- Возможность оставить отзыв
- Подключение аналитики (Яндекс.Метрика, Google Analytics)
- Интеграция с мессенджерами для быстрой связи (WhatsApp, Telegram)

## Дизайн и стиль

**Общие принципы:**
- Минималистичный, чистый дизайн
- Профессиональная цветовая палитра (основной: белый/светло-серый, акцентный: синий/бирюзовый)
- Качественная типографика (читабельные шрифты, иерархия заголовков)
- Достаточное "воздуха" между элементами
- Плавные анимации и переходы

**Типографика:**
- Заголовки: современный геометрический шрифт (например, Montserrat, Roboto)
- Основной текст: легко читаемый шрифт (например, Open Sans, Lato)
- Размеры: адаптивная типографика для разных устройств

**Изображения:**
- Профессиональное фотография учителя
- Возможно использование иконок для преимуществ и тарифов
- Минимум декоративных элементов, акцент на содержании

## Дополнительные рекомендации

1. **Контент:** Тексты должны быть убедительными, но не слишком длинными. Использовать простые и понятные формулировки.
2. **Призывы к действию:** Четкие и заметные кнопки с призывами к действию на каждом этапе.
3. **Доверие:** Добавить элементы, вызывающие доверие (дипломы, сертификаты, отзывы).
4. **Удобство:** Процесс записи на занятия должен быть максимально простым.
5. **Тестирование:** Обязательно протестировать сайт на различных устройствах и браузерах.
